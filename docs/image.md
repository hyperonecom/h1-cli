## h1 image list

List image

### Syntax

```h1 image list | [--recommended] [--all]```

### Examples

#### List all user images

```bash
h1 image list
```

#### List public images recommended by platform

```bash
h1 image list --recommended
```

### Examples


| Name | Default | Description |
| ---- | ------- | ----------- |
| ```--recommended``` |  | Display recommended images |
