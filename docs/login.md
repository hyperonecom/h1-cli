# Specification

# h1 login

Obtain your apiKey

## Syntax

```h1 login | [--username USERNAME] [--password PASSWORD]```
## Example

```bash
h1 login --username user@example.com
```

### Optional arguments

| Name | Default | Description |
| ---- | ------- | ----------- |
| ```--username USERNAME``` |  | Your username |
| ```--password PASSWORD``` |  | Password |
