
# h1 iam project application

Manage applications of the project

## Synopsis

```bash
$ h1 iam project application <options>
```

## Operation options

| Option name               | Description |
| ------------------------- | ----------- |
| ```--command {command}``` | -           |

## Common options

| Option name          | Description                 |
| -------------------- | --------------------------- |
| ```--help```         | Show help message and exit. |
| ```--v, --version``` | Show version and exit.      |
