
# h1

Management for cloud services of HyperOne

## Synopsis

```bash
$ h1 <options>
```

## Common options

| Option name               | Description                 |
| ------------------------- | --------------------------- |
| ```--command {command}``` | -                           |
| ```--help```              | Show help message and exit. |
| ```--v, --version```      | Show version and exit.      |
