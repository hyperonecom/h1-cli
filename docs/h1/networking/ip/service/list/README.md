
# h1 networking ip service list

List networking/ip.service

## Synopsis

```bash
$ h1 networking ip service list <options>
```

## Description

See also:

* [API reference](https://api.hyperone.com/v2/docs#operation/networking_project_ip_service_list)

## Example


### Simple

```bash
h1 networking ip service list \ 
	--project 5f64e2468c71177993874510 \ 
	--ip 5f577a24494c5cfdec7830e5
```
