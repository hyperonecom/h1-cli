
# h1 version

Manage version of CLI

## Synopsis

```bash
$ h1 version <options>
```

## Operation options

| Option name               | Description |
| ------------------------- | ----------- |
| ```--command {command}``` | -           |

## Common options

| Option name          | Description                 |
| -------------------- | --------------------------- |
| ```--help```         | Show help message and exit. |
| ```--v, --version``` | Show version and exit.      |
