
# h1 config

Management of CLI configuration

## Synopsis

```bash
$ h1 config <options>
```

## Common options

| Option name               | Description                 |
| ------------------------- | --------------------------- |
| ```--command {command}``` | -                           |
| ```--help```              | Show help message and exit. |
| ```--v, --version```      | Show version and exit.      |
