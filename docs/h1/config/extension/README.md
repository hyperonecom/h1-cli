
# h1 config extension

Manage extensions of CLI

## Synopsis

```bash
$ h1 config extension <options>
```

## Common options

| Option name               | Description                 |
| ------------------------- | --------------------------- |
| ```--command {command}``` | -                           |
| ```--help```              | Show help message and exit. |
| ```--v, --version```      | Show version and exit.      |
