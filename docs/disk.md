## h1 disk create

Create disk

### Syntax

```h1 disk create | --name NAME --type TYPE --size SIZE [--source-file SOURCE-FILE] [--no-progress]```

### Examples

#### Create 10 GB SSD disk

```bash
h1 disk create --name db-disk --type ssd --size 10
```
